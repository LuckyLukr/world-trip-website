# Cesta kolem světa TRIP

Vytvoř promyšlený web pro Blog.

Hlavní cíl: Publikovat obsah.

Požadované sekce a moduly: Header / navigace, Hero, Kontakt, Galerie / grid, O nás.

Detaily pro sekci Galerie a obrázky: Co má galerie ukázat: Instagram fotky, selfie fotky, fotky krajiny.

Kde to dává smysl, použij navigaci, feature gridy, carousel referencí, gallery grid, CTA banner i mapové / lokační bloky.

Vizuální směr: tón Moderní a minimální; energie layoutu Hravá; styling pozadí Abstraktní tvary; chování headeru Sticky, slít s hero; práce s hero médiem Celoplošný obrázek na pozadí; styl povrchů Elevated cards; kompozice hero Split.

Vyhni se plochému generickému webu. Použij kontrast, vrstvené povrchy a dekorativní background treatment, který odpovídá zvolenému vizuálnímu směru.

Veškerý obsah webu napiš v jazyce Česky.

Nepřekročitelné instrukce: Vytvoř moderní, elegantní a mobil-first web pro projekt „Cesta kolem světa“ — jde o osobní cestovatelský blog AI persony Klára z Valtic, která formou deníkových zápisů sdílí svou fiktivní cestu po světě. Web má působit jako jemný, vizuálně čistý travel blog s důrazem na velké fotografie, čitelnost a emoci. Součástí má být homepage, blog, detail článku, stránka O Kláře, stránka Trasa a stránka O projektu / kontakt. Design doporučuji světlý, vzdušný a prémiový: teplá bílá, písková, šalvějová a tlumená modrá; nadpisy serif, text čistý sans-serif. Tone of voice má být lyrický, srdečný a zvídavý, ale zároveň transparentní — je potřeba jasně uvést, že Klára je AI persona powered by JeanAI.cz. Blog je hlavní obsahový formát a má být připravený na publikaci draftů generovaných agentem přes API s následným lidským schválením před publikací..

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Tailwind CSS is preconfigured through Vite and loaded from `src/tailwind.css`.
